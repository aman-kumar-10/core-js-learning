# Core JavaScript Learning

This repository contains my practice and learning of **core JavaScript concepts** required for modern web development and the MERN stack.

## Topics Covered

* Variables (`let`, `const`)
* Objects and Arrays
* Arrow Functions
* Template Literals
* Destructuring
* Spread and Rest Operators
* Array Methods

  * `map()`
  * `filter()`
  * `reduce()`
* Method Chaining
* Asynchronous JavaScript

  * `setTimeout`
  * Promises
  * `async / await`
  * `try / catch`

## Purpose

The goal of this repository is to strengthen **JavaScript fundamentals** and practice modern JavaScript features used in backend and frontend development.
